problem/data domain

App description

3.What will the users do with the web app? What kind of interaction are you expecting to provide?

Users will need to create their own account for our app and they will need to log in before using our app. There are several ways users can interact with our app. User can search a name directly. When they hover their mouse on the name they want to look up, they can view information relavent to the name such as gender, origin, and meaning, etc. For users don't have a name in mind but want to find one, they are allowed to filter for a name in the app's database based on their preferences. They can filter through categories such as gender, origin, and spelling, etc. to find a name they want. Users are also allowed to bookmark names and create a list of names they want and view it when they want.

4. How will using this app help to solve your chosen problem? That is, how is it a solution to the problem you described?

Our app provides a way that ensures safety and privavy for users obtaining a name they want. Meanwhile, it provides a convenient way for users from different social groups to look for a name based on their needs and preferences. Groups such as immigrants and parents can find a meaningful name they want through our app. Since our app has features that appeal users needs, we provide a way that maximize users freedom and ownership.
